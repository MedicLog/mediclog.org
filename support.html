Help Us Test on More Devices

MedicLog is a one-person project built by an active EMS provider. The core app is tested on a limited set of hardware — but EMS crews run everything from an SE to an Ultra, old iPads to new, iOS and Android side by side.

If you've found MedicLog useful and want to help it work well for everyone, here's what we're missing.

Why hardware matters: Screen sizes, GPU performance, haptic engines, and sensor timing all vary between devices. A timer that fires perfectly on a Series 11 may behave differently on an SE. An iPad mini layout that looks great on a 4th gen Air may need adjustment on a mini screen. Real hardware testing catches things simulators don't.

How to Help

If you have hardware you're not using, or want to contribute to expanding test coverage, the Amazon wishlist has everything we're looking for. No pressure — even one device helps.

Prefer to support development directly? GitHub Sponsors lets you contribute any amount, one-time or monthly. Every bit keeps this project moving.

If something you want to send isn't on the wishlist, reach out via the GitHub support page first so we can coordinate.
