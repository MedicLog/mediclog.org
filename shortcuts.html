1 Getting Started

What are iOS Shortcuts?

iOS Shortcuts is Apple's built-in automation app that lets you create custom workflows. You can trigger actions automatically (like when you receive a text message) or manually with a single tap or voice command.

Why Use Shortcuts with MedicLog?

In emergency situations, every second counts. Shortcuts let you:

  • Auto-start calls when you receive a dispatch SMS
  • Log status updates with a single tap or NFC tag
  • Reduce manual input while wearing gloves or in a moving ambulance
  • Create consistent workflows that work every time

Prerequisites

  • iOS 16 or later (iOS 17+ recommended for best experience)
  • MedicLog installed from the App Store
  • Shortcuts app (pre-installed on all iPhones)

2 Step-by-Step Setup

Let's create a basic automation that starts a new call when you receive a dispatch message.

Open the Shortcuts App

Find and tap the Shortcuts app on your home screen. It has a colorful icon with overlapping squares.

Go to the Automation Tab

Tap Automation at the bottom of the screen. This is where you create triggers that run automatically.

Create New Automation

Tap the + button in the top right corner, then tap Create Personal Automation.

Choose Your Trigger

Scroll down and select Message. Configure it to trigger when you receive a message from your dispatch number containing specific keywords like "Respond to" or "Call at".

Add MedicLog Action

Tap Add Action, then search for "MedicLog". Select the action you want, such as Start New Call or Receive Run Number.

Configure the Action

Some actions have parameters you can fill in. For Receive Run Number, you can pass the message content so MedicLog can extract the address automatically.

Enable "Run Without Asking"

Before saving, toggle off Ask Before Running. This is critical for emergency automations to work instantly without confirmation.

Save and Test

Tap Done to save. Test by having someone send you a message matching your trigger criteria.

Important: For message-based automations, you must specify a sender. iOS requires this for security. Use your dispatch center's phone number.

3 Automation Recipes

Here are ready-to-use automation ideas for common EMS workflows.

Dispatch SMS Auto-Start
Trigger Receive SMS from dispatch number containing "Respond to"
Action Receive Run Number with message content passed as parameter
Result MedicLog opens with a new call, address pre-filled from the dispatch message
Quick Status Update via NFC
Trigger NFC tag scan (place tags on your rig's dashboard)
Action Log Entry with status text (e.g., "On Scene", "Transporting", "At Hospital")
Result Status logged to the active call with a single tap, no screen interaction needed
Voice-Triggered Log Entry
Trigger Say "Hey Siri, log vitals" or add to home screen for one-tap access
Action Log Entry action in MedicLog
Result MedicLog opens ready to record, hands-free activation
End of Call Workflow
Trigger Manual shortcut (add to home screen or widget)
Action End Current Call in MedicLog
Result Call is closed and ready for documentation review
Pro Tip: Create multiple NFC tags with different pre-configured status entries. Label them clearly and mount them where you can tap them easily during a call.

4 Available MedicLog Actions

These are the Shortcuts actions currently available in MedicLog. Search for "MedicLog" in the Shortcuts app to find them.

Start New Call

Begins a new call session in MedicLog. Use this when you're dispatched to a new incident.

StartCallIntent

End Current Call

Closes the active call session. The call data remains saved for later review and export.

EndCallIntent

Log Entry

Adds a new entry to the active call. You can pass text content to pre-fill the entry.

LogEntryIntent

Receive Run Number

Receives dispatch information from an SMS automation. Parses the message to extract run number and address.

ReceiveRunNumberIntent
Note: More Shortcuts actions are coming in future updates. Features like "Start Timer" and "Export PCR" are planned but not yet available via Shortcuts.

5 Troubleshooting

"My automation doesn't run automatically"
Make sure Ask Before Running is turned off in the automation settings. iOS requires confirmation by default for many triggers. Also verify the trigger conditions match exactly (sender number, keywords, etc.).
"MedicLog doesn't appear in Shortcuts actions"
Ensure MedicLog is installed and has been opened at least once. Try restarting your iPhone. If still missing, delete and reinstall MedicLog from the App Store.
"The action runs but MedicLog doesn't open"
Some actions work in the background. If you need MedicLog to open, add an "Open App" action before your MedicLog action, or use actions that require the app to be in the foreground.
"Data from the message isn't passed correctly"
When setting up the action, make sure you're using the Shortcut Input variable (the message content) rather than typing static text. Tap the variable button and select the message body from the trigger.
"My NFC automation stopped working"
NFC automations require your iPhone to be unlocked. If the screen is off, you'll need to tap the NFC tag, then authenticate. Consider using a time-based trigger or manual shortcut as a backup.
"Siri doesn't recognize my shortcut name"
Give your shortcut a simple, distinct name that's easy to pronounce. Avoid names that sound like common phrases. You can also add alternative phrases in the shortcut's settings.
