How to Request a Claude ZDR API Key

Set up Zero Data Retention for HIPAA-compliant use of MedicLog's AI-powered features.

Why ZDR matters for HIPAA compliance MedicLog uses Claude Vision to extract data from photos of cardiac monitors (vitals OCR) and medication packaging (medication OCR). These images are sent to Anthropic's Claude API for processing. Under standard API terms, Anthropic may retain prompts and responses. With Zero Data Retention, your data is not stored at rest after the API response is returned — prompts and outputs are processed in real time and discarded. This means patient data in monitor photos and medication images never persists on Anthropic's servers, which is a key requirement for HIPAA compliance when processing protected health information (PHI). Anthropic also offers HIPAA-ready API access with a signed Business Associate Agreement (BAA) for organizations that need formal HIPAA coverage.

How to request access

  1. Contact Anthropic Sales

    Visit claude.com/contact-sales. Request ZDR if you're an individual provider, or HIPAA-ready API access if you're an agency. Mention that you're using the Claude Messages API for medical image analysis (monitor vitals and medication OCR) in MedicLog.

  2. Sign a BAA (agencies only)

    For HIPAA-ready access, Anthropic will provide a Business Associate Agreement. Individual providers requesting ZDR only do not need a BAA.

  3. Receive your dedicated organization

    Anthropic provisions a dedicated organization with ZDR or HIPAA controls enabled. HIPAA-enabled organizations automatically block API features that aren't covered by the BAA.

  4. Generate an API key

    Create an API key from your new organization at console.anthropic.com. This key inherits the ZDR or HIPAA protections of the organization.

  5. Enter the key in MedicLog

    Open MedicLog → Settings → AI Parsing → enter your API key. All Vision OCR requests will now route through your protected organization.

What's covered

Eligible

  • Messages API (what MedicLog uses)
  • Vision / image input
  • Token counting
  • Prompt caching
  • Extended thinking

Not eligible

  • Batch processing
  • Code execution
  • Files API
  • Console / Workbench usage
  • Claude consumer products (Pro, Max)
Exception: Even with ZDR or HIPAA arrangements, Anthropic may retain data where required by law or to address Usage Policy violations — up to 2 years if a session is flagged.
