## Capture Tips

### iPhone 15 (Physical Device)
- Side button + Volume Up = screenshot
- AirDrop to Mac, rename

### Apple Watch
- Side button + Digital Crown simultaneously
- Screenshot appears in Photos on paired iPhone

### iPad Air 4
- Top button + Volume Up (or Home + Top if applicable)

### Clean Status Bar
- Full battery, good signal, reasonable time (not 4:20 or 12:34)
- Or use Settings → Developer → Status Bar Override

### Recommended
- Light mode for most shots (matches website default)
- Dark mode for the hero comparison shot
- Realistic but clean data (believable vitals, med names)
